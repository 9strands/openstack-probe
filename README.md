# openstack-probe
Probes openstack cloud for: resources without a valid (existing) project / orphaned resources.

## How to
```console
git clone https://github.com/yvarbanov/openstack-probe.git osp-probe
cd osp-probe
vim clouds.yaml
bash quickstart.sh
```
